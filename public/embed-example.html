🌐 Network Override Widget

This page demonstrates how the Network Override Widget can be embedded into any website. The widget provides a floating interface for intercepting and modifying network requests without requiring complex DevTools setup.

📦 Installation

To embed this widget on your website, simply include the script and initialize it:

<!-- Include the widget script --> <script src="https://your-domain.com/network-override-widget.js"></script> <!-- Initialize the widget --> <script> NetworkOverrideWidget.init({ position: 'bottom-right', // Optional: 'bottom-left', 'top-right', 'top-left' autoStart: true // Optional: Start intercepting immediately }); </script>

🚀 Features

  • 🌐 Floating Icon - Unobtrusive widget that stays out of the way
  • 📱 Fullscreen Interface - Expands to show full network override tools
  • 🔄 Live Interception - Captures both fetch() and XHR requests automatically
  • 💾 Persistent Overrides - Saves override configurations across page reloads
  • 📁 File Downloads - Exports override configurations as JSON files
  • 🎯 Visual Indicators - Shows activity and override status in real-time

🔧 Configuration Options

NetworkOverrideWidget.init({ // Widget position position: 'bottom-right', // 'bottom-right', 'bottom-left', 'top-right', 'top-left' // Auto-start interception autoStart: true, // Start intercepting network requests immediately // Custom styling theme: 'default', // 'default', 'dark', 'light' // Override storage persistOverrides: true, // Save overrides to localStorage // Activity notifications showActivityIndicator: true, // Show green pulse on network activity // Callback functions onRequestIntercepted: (request) => { console.log('Intercepted:', request); }, onOverrideApplied: (url, override) => { console.log('Override applied:', url, override); } });
